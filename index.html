Requirements Before You Begin

  • A stable Wi-Fi connection
  • Your Canon printer powered on and in setup mode
  • The printer’s model number
  • A computer (Windows or Mac) ready for configuration
  • The latest Canon printer drivers, which you can download from the official Canon support website

How to Find My Canon Printer on This Computer

Windows:

  • Go to Control Panel → Devices and Printers
  • Look for your Canon printer in the list. If missing, proceed to add it manually.

Mac:

  • Go to System Preferences → Printers & Scanners
  • Check if your Canon printer is listed. If not, click the “+” button to add it.

If your computer cannot detect the printer, it may not be connected to the network. This is when you should confirm your Wi-Fi settings.

How to Connect Canon Printer to Wi-Fi

For printers with a touchscreen display:

  1. Go to Menu → Network Settings → Wireless LAN Setup
  2. Choose your Wi-Fi network from the list
  3. Enter your Wi-Fi password
  4. Wait for a confirmation message that the printer is connected

For printers without a touchscreen:

  1. Press and hold the Wi-Fi button until the light flashes
  2. Within 2 minutes, press the WPS button on your router
  3. Wait until the Wi-Fi light stays solid

Once connected, your printer will be ready to install on your computer.

Installing My Canon Printer to My Computer (Windows)

  1. Download the latest drivers from https://ij.start.canon
  2. Run the setup file and select Start Setup
  3. Follow the prompts to detect your printer
  4. Select your printer model from the detected devices list
  5. Confirm the Wi-Fi connection settings
  6. Complete installation and perform a test print

Installing My Canon Printer to My Computer (Mac)

  1. Download the Canon driver package from https://ij.start.canon
  2. Open the downloaded file and follow the on-screen setup instructions
  3. Go to System Preferences → Printers & Scanners
  4. Click the + button and select your Canon printer
  5. Complete the driver installation and confirm by printing a test page

Troubleshooting Printer Detection Issues

  • Restart both your printer and computer
  • Ensure both devices are connected to the same Wi-Fi network
  • Temporarily disable firewall or antivirus software blocking detection
  • Update your printer firmware from the Canon support site
